# 📖 Buscador de Palabras
### Definición RAE + Sinónimos Lexus + Groq

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `app.py` | Aplicación Streamlit |
| `sinonimos.txt` | Diccionario Lexus — 15.869 entradas |
| `requirements.txt` | Dependencias |
| `README.md` | Esta guía |

## Deploy en Streamlit Cloud

1. Sube los 4 archivos a un repositorio de GitHub
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta tu repositorio
4. Listo — Streamlit instala las dependencias automáticamente

## Uso

1. Ingresa tu **Groq API Key** en el panel izquierdo (gratis en console.groq.com)
2. Escribe una palabra y clic en **Buscar**
3. Ver resultados en 3 tabs: Respuesta completa · RAE · Lexus
